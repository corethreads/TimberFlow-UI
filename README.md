Timber Flow UI
## [Overview]
Timber Flow is a specialized calculation tool designed for carpenters to perform accurate measurements and calculations. This frontend application provides an intuitive interface for all carpentry math requirements.

## [Features]
- [Precise Measurements] Calculate lengths, angles, and dimensions with high accuracy
- [Material Estimation] Estimate timber requirements for various project types
- [Unit Conversions] Seamlessly switch between metric and imperial measurement systems
- [Project Persistence] Save and load calculations for future reference
- [Responsive Interface] Optimized for desktop, tablet, and mobile devices

## [Technology Stack]
- [Framework] React.js
- [State Management] Zustand
- [Styling] Tailwind CSS
- [Build Tool] Vite
- [Package Manager] npm or yarn

## [Getting Started]

### [Prerequisites]
- Node.js version 16 or higher
- npm or yarn package manager

### [Installation]
```bash
# Clone the repository
git clone https://github.com/yourusername/timber-flow.git

# Navigate to frontend directory
cd timber-flow/frontend

# Install dependencies
npm install

# or with yarn
yarn install

# Start development server
npm run dev

# or with yarn
yarn dev
